# SabziBuddy

> Taazi Sabzi, Farm se Ghar Tak

## About

SabziBuddy ek Firebase-powered e-commerce platform hai jo taazi sabzi, fruits, dairy, aur groceries ki home delivery provide karta hai.

## Modules

| Module | Path | Description |
|--------|------|-------------|
| Customer Website | `customer/` | User-facing shopping website |
| Admin Panel | `admin/` | Owner/Manager/Delivery Boy dashboard |
| Docs | `docs/` | Full documentation |

## Tech Stack

- HTML5 + CSS3 + JavaScript (Vanilla)
- Firebase (Auth, Firestore, Storage)
- Leaflet.js (Maps)
- Google Fonts (Outfit, Cormorant Garamond)

## Firebase Project

- Project: `sabzibuddy-website`
- Auth: OTP (customer) + Email/Password (admin)
- Database: Firestore
- Storage: Product images
